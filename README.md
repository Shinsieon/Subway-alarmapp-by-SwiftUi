# Swiftui_WakeUp
Swiftui Application

목적지(지하철역)에 도착하면 벨과 진동을 통해 깨워드립니다!

애플 맵을 통해 위치를 조회하며 목적지와의 거리를 계산하여 지정하신 반경에 도착하면 깨워드려요

걱정없이 지하철에서 잠들어도 됩니다
